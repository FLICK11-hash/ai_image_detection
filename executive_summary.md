# Executive Summary For AI Detector
  This project focused on building an AI image detector that can determine whether a picture was created by artificial intelligence or is a real photograph. 
  Using a dataset containing 60,000 real images and 60,000 AI-generated images, I trained several machine learning models to recognize subtle differences within patterns of color,
  textures, edges, and image frequencies. I also compared traditional machine learning techniques against a neural network and tested how well the detector worked after images were resized, 
  blurred, or compressed. The best-performing model achieved approximately 91% accuracy, demonstrating that statistical image analysis can be an effective way to identify 
  AI-generated content without requiring extremely large or expensive deep learning systems.
